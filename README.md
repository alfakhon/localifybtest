Localify — simple React landing (single-folder)
===============================================

Qanday ishlatish:
1. Node.js o'rnatilgan bo'lishi kerak (Node 18+ tavsiya qilinadi).
2. Terminalda papkaga o'ting:
   npm install
   npm run dev
3. Brauzerda http://localhost:5173 oching (vite default).

Eslatma:
- Barcha fayllar bitta papkada joylashgan (subfolder yo'q).
- Tailwind CDN Play ishlatilgan (https://cdn.tailwindcss.com) shuning uchun build jarayonida ham Tailwind ishlaydi.
- Ikonlar `icons.jsx` ichida inline SVG sifatida berilgan — tashqi ikon kutubxonasiga ehtiyoj yo'q.
